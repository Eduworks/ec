EC UI Framework
===============
Last Revised: September 2016
Developed by Devlin Junker (devlin.junker@eduworks.com)

Notes:
	Detailed Documentation on Developer Site
	
	Javadoc for framework classes available in doc folder
	
	Example application available at svn://svn.eduworks.com/ui/ec.ui.example.application

	To start a new Project from the template (detailed instructions on dev site)
		1. Checkout and build ec and ew projects
		2. Checkout ec.ui.template.application project
		3. Detach ec.ui.template.application from SVN
		4. Rename ec.ui.template.application project and change all references to new project name (pom.xml, index.html)
